## 八、部署建议

### 环境要求
- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.8+

### 启动步骤

**1. 后端启动**
```powershell
cd D:\Axm\Project\学生信息管理系统\backend
mvn spring-boot:run
```

**2. 前端启动**
```powershell
cd D:\Axm\Project\学生信息管理系统\frontend
npm run dev
```

**3. 初始化数据库**
```sql
mysql -u root -p student_management < D:\Axm\Project\学生信息管理系统\database\init.sql
mysql -u root -p student_management < D:\Axm\Project\学生信息管理系统\database\test_data.sql
```

### 生产部署配置
- 修改 JWT_SECRET 环境变量为生产密钥（>=64字符）
- 配置生产数据库连接
- 配置 HTTPS 反向代理（Nginx）
- 修改 MySQL 密码为强密码
