## 项目目标

将本地学生信息管理系统部署到互联网，完全免费。

### 部署架构

```
用户浏览器
    ↓ HTTPS
Vercel (前端)  →  https://xxx.vercel.app
    ↓ API 请求
Railway (后端) →  https://student-api.onrender.com
    ↓
Neon (PostgreSQL) →  postgresql://xxx
```

-- 自动创建，无需手动建库

-- 用户表

    major VARCHAR(50) NOT NULL,
    grade VARCHAR(20) NOT NULL,
    teacher_id BIGINT,
    student_count INT DEFAULT 0,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 学生表

    id BIGSERIAL PRIMARY KEY,

    email VARCHAR(50),
    id_card VARCHAR(20),
    address VARCHAR(200),
    birth_date DATE,
    enrollment_date DATE,

    avatar VARCHAR(200),

-- 课程表
CREATE TABLE IF NOT EXISTS course (
    id BIGSERIAL PRIMARY KEY,
    course_name VARCHAR(100) NOT NULL,
    course_code VARCHAR(20),
    credit DECIMAL(3,1) NOT NULL,
    teacher_id BIGINT,
    teacher_name VARCHAR(50),
    semester VARCHAR(20),
    class_hours INT,
    description TEXT,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 成绩表
CREATE TABLE IF NOT EXISTS score (
    id BIGSERIAL PRIMARY KEY,
    student_id BIGINT NOT NULL,
    student_no VARCHAR(20),
    student_name VARCHAR(50),
    course_id BIGINT NOT NULL,
    course_name VARCHAR(100),
    score DECIMAL(5,2),
    credit DECIMAL(3,1),
    semester VARCHAR(20),
    exam_type SMALLINT DEFAULT 1,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 学籍异动表
CREATE TABLE IF NOT EXISTS status_change (
    id BIGSERIAL PRIMARY KEY,
    student_id BIGINT NOT NULL,
    student_no VARCHAR(20),
    student_name VARCHAR(50),

-- 奖惩记录表
CREATE TABLE IF NOT EXISTS reward_punish (
    id BIGSERIAL PRIMARY KEY,
    student_id BIGINT NOT NULL,
    student_no VARCHAR(20),

    type SMALLINT NOT NULL,
    title VARCHAR(100) NOT NULL,
    content TEXT,
    record_date DATE,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 奖学金表
CREATE TABLE IF NOT EXISTS scholarship (
    id BIGSERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    amount DECIMAL(10,2),
    level VARCHAR(20),
    description TEXT,
    create_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP

-- 奖学金申请表
CREATE TABLE IF NOT EXISTS scholarship_apply (
    id BIGSERIAL PRIMARY KEY,
    student_id BIGINT NOT NULL,

-- 创建索引

### 任务 8：创建 Railway 配置文件

```dockerfile
FROM eclipse-temurin:21-jdk-alpine AS builder
WORKDIR /app
COPY pom.xml .
COPY src ./src
RUN apk add --no-cache maven
RUN mvn package -DskipTests

FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY --from=builder /app/target/*.jar app.jar
EXPOSE 8080

```

**文件 3**：`D:\Axm\Project\学生信息管理系统\render.yaml`（Railway Nixpacks 配置）

```yaml
services:
  - type: web
    name: student-api
    runtime: docker
    repo: https://github.com/你的用户名/student-management
    dockerfilePath: backend/Dockerfile
    region: singapore
    plan: starter
    envVars:
      - key: DATABASE_URL
        fromDatabase:
          name: student-db
          property: connectionString
      - key: JWT_SECRET
        generateValue: true
      - key: ENABLE_SWAGGER
        value: "false"
      - key: PORT
        value: "8080"
      - key: PROFILE
        value: "default"
    healthCheckPath: /auth/login
    autoDeploy: true

databases:
  - name: student-db
    databaseName: student_management
    plan: starter
    pgVersion: 15
```

**或者使用 Nixpacks（无需 Dockerfile）**：创建 `D:\Axm\Project\学生信息管理系统\backend\nixpacks.toml`：
